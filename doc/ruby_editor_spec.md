# ルビタグ編集ツール 仕様書

## 1. 画面構成
- 画面遷移はない
- 以下の3エリアがある
    - 左：ソーステキストエリア
    - 右：プレビューエリア
    - 下：ボタンエリア

### 1-1. ソーステキストエリアの内容
- **タイトル表記**：ソース
- テキストを編集できるエリア
- **初期テキスト**：「ここに<ruby=もじ>文字</ruby>をいれてください」

### 1-2. プレビューエリアの内容
- **タイトル表記**：プレビュー
- ソーステキストの内容を実際にルビ表示でプレビューする
- プレビューエリアのテキストは編集できない

### 1-3. ボタンエリアの内容
- **ボタン**：ソースをコピー
- **ボタン**：ひとつ戻る
- **ボタン**：クリア

---

## 2. 動作仕様

### 2-1. ソーステキストエリアの動作
- テキストエリアでマウスダウンした瞬間、初期テキストがあれば初期テキストは消える
- ソーステキストの文字を選択してマウスアップした瞬間に下記の動作をする
    - 選択状態を解除する
    - 選択されていた文字の左側に `<ruby=>` を入れる
    - 選択されていた文字の右側に `</ruby>` を入れる
    - テキストカーソルを `<ruby=` の右側に置く

### 2-2. プレビューエリアの動作
- ソーステキストに変化が起きたら、プレビューをリアルタイムに更新する

### 2-3. ボタンエリアの動作
- **ソースをコピーボタン**：ソーステキストをクリップボードにコピーする
- **ひとつ戻るボタン**：ソーステキストの変化を元に戻す。（最大5回前まで戻れるように履歴を管理する）
- **クリアボタン**：ソーステキストエリアを空にする
